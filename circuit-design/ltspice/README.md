# LTspice Circuit Simulation

This directory contains LTspice simulations of the lag compensator control system implemented using analog circuits.

The objective of this section is to verify that the circuit implementation behaves consistently with the theoretical transfer function design and MATLAB/Simulink simulations.

The LTspice simulations include:

- Uncompensated control system circuit
- Lag compensated control system circuit
- Step response analysis
- Ramp response analysis

These simulations demonstrate how the lag compensator affects the system behavior in an analog implementation.

The results are compared with the theoretical calculations and MATLAB/Simulink simulations presented in other sections of this project.

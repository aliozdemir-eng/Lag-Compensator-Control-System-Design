# System Simulation

This directory contains simulation studies of the control system before and after the lag compensator is introduced.

The goal of these simulations is to evaluate how the lag compensator affects the system response.

The following analyses will be included:

- Step response comparison
- Ramp response comparison
- Time-domain performance evaluation
- Verification of steady-state error improvement
- Observation of transient response behavior

Both uncompensated and compensated systems will be analyzed in order to clearly observe the effect of the lag compensator on system dynamics.

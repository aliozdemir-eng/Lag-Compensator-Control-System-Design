# MATLAB / Simulink Analysis

This directory contains the MATLAB / Simulink based analysis of the lag compensator control system.

The purpose of this section is to verify the theoretical calculations by comparing the uncompensated and lag compensated systems in simulation.

The following materials will be included in this directory:

- Simulink model screenshots
- Step response comparison plots
- Ramp response comparison plots
- Numerical simulation results
- Observations on overshoot, settling time, and steady-state error

These simulation results are used to confirm the effect of the lag compensator on both transient response and steady-state performance.

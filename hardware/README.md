# Hardware Implementation

This section presents the physical implementation of the lag compensator control system.

After completing the simulation and PCB design stages, the circuit was implemented and tested using real electronic components.

Two hardware implementations were constructed:

• Breadboard prototype  
• Perforated copper board implementation

These implementations were used to verify the practical behavior of the designed control system.

The hardware stage demonstrates the transition from simulation and PCB design to a working physical circuit.

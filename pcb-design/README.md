# PCB Design

This directory contains the PCB design stage of the lag compensator control system project.

The purpose of this section is to show how the analog control circuit was transferred from simulation into a manufacturable and physically implementable board design.

The PCB design stage includes:

- Uncompensated system PCB layout
- Lag compensated system PCB layout
- 2D board views
- 3D board views
- Component placement overview

This stage demonstrates the practical realization of the control system in hardware form.
